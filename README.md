# Bank Scraper

A TypeScript project for bank scraping.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Development:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

4. Start:
```bash
npm start
```

## Linting

- Run linter:
```bash
npm run lint
```

- Fix linting issues:
```bash
npm run lint:fix
```
